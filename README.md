# Smart Petrol Cost Calculator with BUDI MADANI Rebate

A mobile application developed for ICT602 Mobile Technology assignment.
This app estimates total petrol costs in Malaysia and applies the BUDI MADANI 
fuel subsidy for eligible users.

## Features
- Select petrol type (RON95, RON97, Diesel)
- Enter petrol price per liter
- Enter fuel usage in liters
- Check BUDI MADANI eligibility
- Calculate total petrol cost
- Calculate BUDI MADANI rebate (RON95 only)
- Calculate total savings

## How to Calculate
- Total Petrol Cost = Fuel Usage × Petrol Price per Liter
- BUDI Rebate = Fuel Usage × RM1.99 (RON95 + eligible only)
- Total Saving = Total Petrol Cost – BUDI Rebate

## Sample Calculation
| Input | Value |
|-------|-------|
| Petrol Type | RON95 |
| Price per Liter | RM4.27 |
| Fuel Usage | 40L |
| BUDI Eligible | Yes |

**Result:**
- Total Cost: RM170.80
- BUDI Rebate: RM79.60
- Total Saving: RM91.20
